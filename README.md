# Useful commands


Install vsce:
```
npm install -g @vscode/vsce
```

Build extension:
```
vsce package
```

Install extension:
```
code --install-extension matugen-watcher-0.0.1.vsix  
```

Temporary fix for watcher- start vs code via:
```
code --extensionDevelopmentPath=~/vscode-theme-build/matugen-watcher
```
